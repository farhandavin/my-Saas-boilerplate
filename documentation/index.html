Documentation

Welcome to the Ultimate SaaS Boilerplate documentation. This guide will help you set up, configure, and launch your SaaS application in minutes.

Note: This boilerplate uses the PERN stack (PostgreSQL, Express, React, Node.js).

1. Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js (v18 or higher)
  • PostgreSQL (Running locally or a cloud instance like Supabase/Neon)
  • Stripe Account (For payments)
  • Google Cloud Project (For OAuth & Gemini AI)

2. Installation

We've included an automated script to make setup effortless.

Option A: Automated Setup (Recommended)

node setup-project.js

This command installs dependencies for Root, Backend, and Frontend automatically.

Option B: Manual Installation

# 1. Install Root Dependencies
npm install

# 2. Install Backend
cd backend && npm install
cp .env.example .env

# 3. Install Frontend
cd ../frontend && npm install

3. Configuration

Open the .env file in the backend/ directory. You must fill in these values:

  • DATABASE_URL: Your PostgreSQL connection string.
  • JWT_SECRET: Run openssl rand -base64 32 in your terminal to generate one.
  • CLIENT_URL: http://localhost:5173 (for local development).

4. Database Setup

Once your .env is ready, initialize the database schema.

cd backend
npx prisma generate
npx prisma db push

5. Stripe Setup (Important)

To enable subscriptions, sync Stripe with your code:

  1. Create two products in Stripe Dashboard: Pro and Team.
  2. Copy the Price ID (e.g., price_1Xy...) for both.
  3. Update backend/src/controllers/paymentController.js with these IDs.
  4. Update frontend/src/pages/Dashboard.jsx with these IDs.
  5. Run Stripe CLI to get your Webhook Secret:
stripe listen --forward-to localhost:5001/api/webhook

Paste the whsec_... secret into your .env file.

6. Auth & AI Keys

Google OAuth

Create a project in Google Cloud Console, enable OAuth, and set the redirect URI to:

http://localhost:5001/api/auth/google/callback

Gemini AI

Get your API key from Google AI Studio and paste it into GEMINI_API_KEY in your .env file.

7. Running the App

Start both frontend and backend with a single command:

npm run dev
  • Frontend: http://localhost:5173
  • Backend: http://localhost:5001

8. Feature Usage

  • AI Generator: Go to "AI Tools" in Dashboard. Prompts are processed via Google Gemini.
  • Team Management: Create organizations and manage members via the "Team" tab.
  • Billing: Test upgrades using Stripe Test Cards (e.g., 4242 4242...).

9. Deployment

The project includes vercel.json for easy deployment.

  • Frontend: Deploy to Vercel/Netlify. Set VITE_API_URL to your backend URL.
  • Backend: Deploy to Render, Railway, or Vercel. Ensure all environment variables are set in the cloud dashboard.
